Visual Guide: Top K Frequent Elements (Bucket Sort)

Array: [1, 1, 1, 2, 2, 3] | K = 2

The "Aha!" Moment: Don't sort the counts. Create an array where the index is the frequency. Put the numbers inside the correct bucket. Then read the array backwards. It takes O(N) time.

Step 1: Count Frequencies (Tally Sheet)

2 2
3 1

Step 3: Scan Backwards (Start from Index 6)

  • Index 3: Found 1. Add to results. Res = [1]. Have we found k=2 items yet? No.
  • Index 2: Found 2. Add to results. Res = [1, 2]. Have we found k=2 items yet? YES.

Stop and Return: [1, 2].
