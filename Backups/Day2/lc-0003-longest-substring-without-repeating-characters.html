Visual Guide: Longest Substring Without Repeats

String: "abcabc"

The "Aha!" Moment: Expand R to capture territory. If R hits a letter already in our Set, crunch L forward to throw away old letters until the duplicate is gone.

Step 1: Grow Window

Expand R (Right). Add 'a', 'b', then 'c'. Max Length = 3.

LabRcabc
Set: { 'a', 'b', 'c' }

Step 2: Collision!

R moves to index 3 ('a'). This is already in the set! We must shrink.

LabcRabc

Cannot add! 'a' is a duplicate.

Step 3: Crunch L Forward

Move L forward by 1. Remove the old 'a' from the Set. Now the window is valid again.

aLbcRabc
Set: { 'b', 'c', 'a' }
