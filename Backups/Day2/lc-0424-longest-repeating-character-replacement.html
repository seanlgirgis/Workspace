Visual Guide: Repeating Character Replacement

String: "AABABBA" | Max Replacements (K): 1

The "Aha!" Moment: The number of elements in the window that MUST be replaced is: Size of Window - Frequency of Most Common Element. If that number exceeds k, the window is invalid.
Window Size - Max Frequency ≤ k

Step 1: Grow the Window

AABABBA

Window: AABA

L=0, R=3 (Size: 4)
Counts: A=3, B=1
Max_Freq: 3 (Leader is 'A')
4 - 3 = 1 ≤ 1 (Valid!)
max_len = 4

Step 2: Add 'B' (Index 4)

AABABBA

Window: AABAB

L=0, R=4 (Size: 5)
Counts: A=3, B=2
Max_Freq: 3
5 - 3 = 2 > 1 (INVALID!)

Step 3: Shift Window Right

Because it's invalid, increment L. Note the window physically shifts rightward but stays size 4.

AABABBA

Window: ABAB

L=1, R=4 (Size: 4)
Counts: A=2, B=2
Max_Freq: 3 (Historical record kept)
4 - 3 = 1 ≤ 1 (Valid!)
max_len = 4
