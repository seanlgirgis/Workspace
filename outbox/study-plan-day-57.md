# Day 57 — Company A: Full Preparation Day

**Theme:** Week 9 is company-specific. Today you prepare deeply for Company A's interview format. Tomorrow you simulate it.

---

## Daily Maintenance (25 min)

**LC — Pattern rotation (2 problems, timed):**
- Pick the 2 patterns most likely to appear at Company A (based on JD or recruiter intel)
- If unknown: default to Medium DP + Medium Graph
- LC example: LC #198 House Robber + LC #200 Number of Islands

**SQL — Finance or data stack specific to Company A:**
- If Snowflake shop: write Time Travel + Stream query
- If Databricks: write Delta MERGE + OPTIMIZE query
- If standard SQL: write a complex window function + CTE pipeline
